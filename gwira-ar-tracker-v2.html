G Wira AR Tracker

One-time setup · 10 minutes · No terminal needed

```
1

Create a free Firebase account

Go to firebase.google.com → Sign in with Google → Click "Get started"

2

Create a new project

Click "Add project" → Name it gwira-ar → Disable Google Analytics → Click Create

3

Enable Firestore Database

Left sidebar → Firestore Database → "Create database" → "Start in test mode" → Choose asia-southeast1 (Singapore) → Done

4

Get your config keys

Project home → Click the Web icon (</>) → Register app → Copy the config block:

const firebaseConfig = { ``` apiKey: “AIzaSy…”, authDomain: “gwira-ar.firebaseapp.com”, projectId: “gwira-ar”, storageBucket: “gwira-ar.appspot.com”, messagingSenderId: “123456…”, appId: “1:123456:web:abc123” };
```
5

Paste config into this file

Open gwira-ar-tracker.html in Notepad or VS Code → Find the comment that says:

// ⬇️ PASTE YOUR FIREBASE CONFIG HERE

Replace the placeholder firebaseConfig with your real one → Save → Refresh the page ✅

🎉

Done! No terminal, no npm, no server.

Upload to GitHub Pages and it syncs across all devices automatically. Phone, PC, iPad — all connected.
