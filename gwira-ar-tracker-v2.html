G Wira AR Tracker

One-time setup · 10 minutes · No terminal needed

1

Create a free Firebase account

Go to firebase.google.com → Sign in with Google → Click "Get started"

2

Create a new project

Click "Add project" → Name it gwira-ar → Disable Analytics → Create

3

Enable Firestore Database

Sidebar → Firestore Database → "Create database" → "Start in test mode" → asia-southeast1 (Singapore)

4

Get your config keys

Project home → Web icon </> → Register app → Copy the config:

const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "gwira-ar.firebaseapp.com", projectId: "gwira-ar", storageBucket: "gwira-ar.appspot.com", messagingSenderId: "123456...", appId: "1:123456:web:abc123" };
5

Paste config into this file

Open gwira-ar-tracker-v2.html in Notepad or VS Code → Find PASTE_YOUR_API_KEY → Replace with your real config → Save → Refresh ✅

🎉

Done! No terminal, no npm, no server.

Upload to GitHub Pages — syncs across phone, PC, and iPad automatically via Firebase.

Need help? WhatsApp your developer 😄
